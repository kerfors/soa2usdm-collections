SoA2USDM Collections

Machine-actionable Schedule of Activities (SoA) extractions from clinical trial protocols.

This repository holds SoA extraction, resolution, and consolidation outputs produced with the soa2usdm pipeline. The usdm_data collection mirrors the data4knowledge/usdm_data protocol set. Source protocol PDFs are fetched from ClinicalTrials.gov and are not stored here.

The collection index is generated by the pipeline (from studies_protocols.xlsx plus discovered outputs) and is the single source of truth for the study list, per-layer pipeline status, and links to each protocol's consolidated SoA, resolved views, extraction JSON, and Excel intermediates.

Pipeline layers

  • Layer 1 — Extraction: SoA table(s) transcribed from the protocol PDF to soa-table-extraction JSON (single-pass, or a PDF→Excel→JSON path for large tables).
  • Layer 2 — Resolve: IDs, hierarchy, and relationships derived per table.
  • Layer 3 — Consolidate: cross-table integration, activity matching, annotation dedup.
  • Visualize: consolidated HTML for human review against the source SoA.
